Χρησιμοποιούμε το Shapiro-Wilk test για να ελέγξουμε κατά πόσο οι τιμές ενός feature τείνουν να αποτελούν κανονική κατανομή. 
Οι εικόνες με τα αποτελέσματα περιλαμβάνονται σε φακέλους. Οι φάκελοι έχουν τα ονόματα των χαρακτηριστικών που αντιπροσωπεύουν.
Εντός των φακέλων οι εικόνες έχουν ονομασία της μορφής {feature}_{m}, όπου feature το εν λόγω χαρακτηριστικό & m=2,3,4,5,6,7,8,9.
Σε κάθε τέτοια εικόνα παρουσιάζονται τα αποτελέσματα του normality test για τη συγκεκριμένη τιμή του m & για όλους τους generators.
