main-code
Η παρούσα διπλωματική εργασία στηρίχτηκε στις αρχικές μελέτες και τα αποτελέσματα της διπλωματικής εργασίας του κ. Καρανίσα Χρήστου με τίτλο "(ΝΑ ΣΥΜΠΛΗΡΩΘΕΙ)".
Από την εργασία αυτή προέρχονται τα αρχεία που βρίσκονται στο φάκελο main-code. Με τα αρχεία αυτά μπορεί να δημιουργήσει κανείς scale free γράφους, καθώς και να δημιουργήσει
dataset με τις τιμές διαφόρων χαρακτηριστικών αυτών. Οι generators και τα χαρακτηριστικά (features) αναφέρονται αναλυτικά στην ανωτέρω εργασία.
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Features Plotting
Δημιουργήθηκαν Dataset για n=100 κόμβους και τιμές του m=2,3,4,5,6,7,8,9. Για κάθε τιμή του m δημιουργήθηκαν 5000 γράφοι για κάθε generator. Στο φάκελο αυτό βρίσκουμε τη γραφική
παράσταση κάθε feature για κάθε τιμή του m.
